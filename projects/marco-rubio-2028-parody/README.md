# Marco Rubio 2028 Parody Prototype

This is a local clickable web prototype for a satirical, non-official parody campaign microsite concept. It is not affiliated with Marco Rubio, any campaign, political committee, party, or election operation.

The prototype intentionally avoids donation, voting, volunteer, petition, email-signup, or campaign-persuasion flows. Its visible copy labels the site as parody.

## What Works

- Vite + React single-page prototype.
- Anchored navigation for Home, Record, Priorities, Press Notes, and About.
- Gallery modal with the image assets used in the page.
- Clickable Record, Priorities, and Press Notes states.
- Responsive desktop and mobile layout.

## Local Commands

```bash
npm install
npm run dev
npm run build
```
